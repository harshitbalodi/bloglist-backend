I welcome you to the bloglist api endpoints

Blog list provide 4 api endpoints
  • User
  • Login
  • Blog
  • test

User

To create user send post request to api/users with name, password and username, username should be of at least three characters

Login

For login send post request to api/login with username and password

Blog

for the point /api/blogs

request endpoint Purpose
GET / return all the blogs in the database
GET /:id/comments returns the comments on the particular blog'id id
POST / to add new blog, Only authorised user with bearer token can add blogs
POST /:id/comments To comment on the blog,Only authorised user with bearer token can comment on the blog
DELETE /:id To delete the blog, Only authorised user with bearer token can delete there own blog.
PUT /:id To like the blog, any authorised user with bearer token can like the blog.

Test

Note: This will only work when server is running on test mode

This will reset the testing database for new testing round, end point for reset is api/test/reset
